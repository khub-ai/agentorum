# Use Cases

This folder contains detailed use case examples for Agentorum, each with sample configurations and representative debate transcripts.

Each use case includes:
- **Configuration** — participant roles, mandates, and LLM assignments
- **Sample transcript** — an annotated excerpt showing how the debate unfolds
- **Output** — the structured summary, data tables, and charts produced
